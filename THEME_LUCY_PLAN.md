# 테마 기획서 — 사이버펑크 '루시' 무드 (Edgerunners)

> **상태: 기획만 완료 (2026-07-20). 구현 전 — §8 확정 대기.**
> 요구의 핵심: **파일 하나 지우면 현 사이트(시네마틱 다크·앰버)로 즉시 복귀**하는 구조. §2가 그 답이다.
> 구현 = `assets/theme-lucy.css` 생성(§7 초안 그대로) + `index.html`에 `<link>` 1줄. 그게 전부다.

---

## 1. 무드 해독 — 포스터에서 색 시스템으로

루시 포스터의 색은 4층이다:

| 포스터 요소 | 추출 색 | 사이트에서 맡을 역할 |
|---|---|---|
| 밤하늘·시티 배경 (딥 인디고) | `#0D0B1F` | 페이지 베이스(--bg) — 현 무채 흑을 남보라 밤으로 |
| 네온 시티라이트 (마젠타) | `#FF3DA5` / `#FF7AC6` | **주 액센트**(--accent/strong) — 앰버의 자리를 승계 |
| 루시의 머리칼 (라벤더 화이트) | `rgba(244,240,255,.92)` | 본문 텍스트 — 순백 대신 라벤더 틴트 |
| CYBERPUNK 로고 (애시드 그린) | `#52F55C` | **Lv.10 만렙 신호** — 노랑의 자리를 승계 |

무드의 본질은 "네온을 많이 쓰는 것"이 아니라 **어둠은 더 깊게(남보라), 강조는 더 쨍하게(마젠타)** — 절제 원칙(§2-10)은 색만 바뀐 채 그대로 유지한다.

## 2. 복귀 가능 아키텍처 (요구사항의 핵심)

```
index.html:  <link rel="stylesheet" href="assets/style.css">
             <link rel="stylesheet" href="assets/theme-lucy.css">  ← 추가되는 유일한 줄
```

- **모든 오버라이드는 `assets/theme-lucy.css` 한 파일에만** 담는다. `style.css`는 단 한 글자도 수정하지 않는다.
- **복귀 = `theme-lucy.css` 파일 삭제** (남은 `<link>`는 404로 조용히 무시됨 — 사이트는 완전한 현 상태로 렌더). 또는 `<link>` 줄 삭제/주석.
- 테마는 `.home` 스코프만 건드림 → **index만 물들고 프로젝트 페이지 5개는 무영향**(각자 고유 액센트 유지 — "무대만 네온, 작품은 그대로").
- 이 md(기획서)는 문서일 뿐 사이트에 로드되지 않는다 — 삭제해도 무방하나, 복귀 스위치는 위의 css 파일이다.

## 5. 강도 3단계 — 어디까지 입힐 것인가

| 레벨 | 내용 | 판단 |
|---|---|---|
| **L1 (기본 권장)** | §3+§4 색 스왑만 | 무드의 80%. 가독성·절제 원칙 무손상 |
| L2 (선택) | 네온 글로우 3곳 한정 — 히어로 `em` 텍스트 글로우, Lv.10 바 글로우, 카드 hover 보더 글로우 | "네온사인" 질감. §7 초안에 주석 블록으로 포함 — 주석만 풀면 켜짐 |
| L3 (비권장) | 시티라이트 그라디언트 배경·글리치 애니메이션 | 과함 — 문서 가독성과 채용 톤을 해침. 기획에서 기각 |

## 6. 리스크와 운용 (정직하게)

- **채용 톤 트레이드오프**: 앰버(차분한 시네마틱 문서) → 마젠타(강한 개성). 힙한 조직엔 플러스, 보수적 조직엔 리스크.
  → 그래서 §2의 **스위치 구조**가 보험이다: 제출 상대에 따라 파일 하나로 A/B 운용 가능.
- **의미 재배선 주의**: "노랑=만렙"이 "그린=만렙"으로 바뀐다(로고 그린 승계라 서사는 오히려 좋아짐). patch 레드는 네온 레드로 유지 — "실패" 판정의 의미 불변.
- 서체는 불변(무드는 색으로만). 히어로 회전·캐러셀 JS 무관(색만 바뀜).
- 프로젝트 페이지와의 단절감: index(네온 남보라) ↔ 페이지(각자 다크). 전환 시 톤 점프가 현행보다 커짐 — 허용 범위(둘 다 다크)나, 신경 쓰이면 §8-1에서 범위 확대 논의.

## 7. 구현 초안 — `assets/theme-lucy.css` (붙여넣기용 전문)

```css
/* ============================================================
   THEME — Cyberpunk 'Lucy' (Edgerunners)
   스위치: 이 파일을 삭제하면(또는 index.html의 link 한 줄을 지우면)
   사이트는 시네마틱 다크(앰버)로 즉시 복귀한다. style.css는 무수정.
   근거·매핑: THEME_LUCY_PLAN.md
   ============================================================ */

/* ── 토큰 스왑 (index 전용 .home 스코프 — 프로젝트 페이지 무영향) ── */
.home{
  --bg:#0D0B1F;                          /* 미드나잇 인디고 (포스터 밤하늘) */
  --surface:#161233;
  --surface-2:#090714;
  --border:rgba(186,158,255,.14);        /* 라벤더 헤어라인 */
  --text:rgba(244,240,255,.92);          /* 라벤더 화이트 (루시의 머리칼) */
  --text-sub:rgba(222,212,246,.62);
  --observed:rgba(222,212,246,.60);
  --accent:#FF3DA5;                      /* 네온 마젠타 (시티라이트) */
  --accent-strong:#FF7AC6;
  --accent-soft:rgba(255,61,165,.15);
  --patch:#FF5464; --patch-soft:rgba(255,84,100,.18);
}

/* ── 하드코딩 재선언 (§4 목록) ── */
.toc-bar.is-scrolled{background:rgba(13,11,31,.82)}
.cr-bgwrap::after{
  background:linear-gradient(180deg,
    rgba(13,11,31,.55) 0%,
    rgba(13,11,31,.32) 34%,
    rgba(13,11,31,.72) 76%,
    rgba(13,11,31,.96) 92%,
    var(--bg) 100%);
}
.cr-card__meta{background:linear-gradient(transparent, rgba(11,8,26,.82))}
.cr-title::before{
  background:radial-gradient(ellipse 64% 60% at 50% 50%,
    rgba(16,11,38,.86) 0%, rgba(16,11,38,.62) 55%, rgba(16,11,38,0) 80%);
}
.cr-chip--kind{background:#F4F0FF; color:#17122E}
.chip--kind{color:#0D0B1F; background:rgba(244,240,255,.94)}
.chip--kind::before{background:#0D0B1F}
.mk{background:rgba(255,61,165,.16)}
.skill--proof .lv{background:rgba(255,61,165,.12)}
.skill--max .bar i.on{background:#52F55C}                 /* 만렙 = 엣지러너 로고 그린 */
.skill--max .lv{color:#52F55C; background:rgba(82,245,92,.16)}

/* ── L2 네온 글로우 (선택 — 켜려면 주석 해제, §5) ──
.cr-kick .lead em{text-shadow:0 0 26px rgba(255,61,165,.5)}
.skill--max .bar i.on{box-shadow:0 0 8px rgba(82,245,92,.35)}
.cards .card:hover{box-shadow:0 16px 40px rgba(0,0,0,.5), 0 0 0 1px rgba(255,61,165,.35)}
*/
```

`index.html` head 수정 1줄 (style.css 링크 바로 아래):

```html
<link rel="stylesheet" href="assets/theme-lucy.css">  <!-- 루시 테마 — 이 줄(또는 파일)을 지우면 원래 무드로 복귀 -->
```

## 8. 검증 계획 (구현 후)

토큰 반영(computed bg/accent) · 하드코딩 10곳 오버라이드 확인 · **프로젝트 페이지 5개 무영향**(body 색 불변) ·
대비 재실측(마젠타 링크·그린 배지) · 375px·콘솔 0 · **복귀 테스트: 파일 삭제 → 현 사이트와 픽셀 동일** → 파일 복원.
