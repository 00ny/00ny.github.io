# 첫 화면 문구(헤드라인) 바꾸기

사이트를 열면 가장 먼저 보이는 **큰 문장**입니다. 지금은 **3개의 문장이 3.4초 간격으로 차례대로 바뀌고, 마지막 문장에서 멈추는** 구조입니다.
자동 재생이 끝나면 문장 **양옆에 화살표(‹ ›)** 가 나타나, 방문자가 이전·다음 문장을 직접 다시 볼 수 있습니다(새로고침 불필요).
아래 순서만 따라 하면 됩니다. 프로그래밍 지식은 필요 없습니다.

---

## 1. 어디를 고치나요 — 두 곳입니다

**`index.html`** 파일을 열고 (메모장·VS Code·GitHub 웹 편집기 등 아무거나) `Ctrl+F` / `⌘+F` 로 찾으세요.

### ① 회전 문장 목록 — `SENTS` 를 찾으세요 (파일 아래쪽)

```js
const SENTS = [
  "기획은 <em>설득의 학문</em>이라 생각합니다.",
  "<em>기획의도</em>가 가장 중요하다고 배웠습니다.",
  "때문에, 항상 <em>'왜'라는 질문</em>에 대비하는<br>기획자 배현입니다."
];
```

- 따옴표 `"…"` **안의 글자만** 바꿉니다. 문장 순서 = 화면에 나오는 순서, **마지막 문장에서 멈춥니다.**
- 문장을 **추가**하려면 줄을 복사해 쉼표(,)로 이어 붙이고, **줄이**려면 줄을 지우면 됩니다(2문장도 동작).
- 회전을 **끄고 한 문장만** 쓰려면: 배열에 문장 1개만 남기세요.

### ② 기본(첫) 문장 — `heroLead` 를 찾으세요 (파일 위쪽)

```html
<span class="lead" id="heroLead">기획은 <em>설득의 학문</em>이라 생각합니다.</span>
```

이건 페이지가 열리자마자 보이는 문장입니다. **SENTS의 1번 문장과 똑같이** 맞춰 두세요.
(자바스크립트가 꺼진 환경·애니메이션 축소 설정 사용자에게는 이 문장만 보입니다.)

---

## 2. 꾸미기 도구 (두 곳 모두 동일)

- **줄 바꾸기** → 바꾸고 싶은 자리에 `<br>` 를 넣습니다.
- **주황색 강조** → 강조할 말을 `<em>` 과 `</em>` 로 감쌉니다. 예: `<em>설득의 학문</em>`

`<span class="who"> … </span>` 는 그 아래 작은 이름·직군 줄입니다. 여기도 글자만 바꾸면 됩니다.

---

## 3. 바꾸는 예시

**속도 조절** — `const HOLD = 3400` 의 숫자(밀리초)를 바꾸면 문장이 머무는 시간이 바뀝니다(3400 = 3.4초).

**문장 2개로 줄이기**
```js
const SENTS = [
  "기획은 <em>설득의 학문</em>이라 생각합니다.",
  "항상 <em>'왜'</em>에 대비하는 기획자 배현입니다."
];
```

**회전 없이 한 문장만**
```js
const SENTS = [
  "플레이어의 선택을 설계합니다"
];
```
(이때 ② 기본 문장도 같은 문장으로 맞춰 주세요.)

---

## 4. 저장하고 반영하기

1. 고친 뒤 파일을 **저장**합니다.
2. 저장소에 **올리면**(commit → push) 1~2분 뒤 https://00ny.github.io 에 자동으로 반영됩니다.
   - GitHub 웹에서 편집했다면 "Commit changes" 버튼만 누르면 됩니다.

---

## 5. 주의사항 (딱 3가지)

- ✅ 따옴표 `"…"` 안의 **글자만** 바꾸기 — 줄 끝의 쉼표(,)와 대괄호 `[` `]` 는 그대로
- ⚠️ 태그는 **짝**으로 — `<em>` 을 열었으면 꼭 `</em>` 로 닫기, `<br>` 는 닫지 않아도 됩니다
- ⚠️ 마지막 문장 뒤에는 쉼표를 붙이지 않아도 됩니다(붙여도 동작은 합니다)

혹시 깨졌다면, 위 예시 중 하나를 **그대로 복사해 붙여넣으면** 돌아옵니다.
