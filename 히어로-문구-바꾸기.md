# 첫 화면 문구(헤드라인) 바꾸기

사이트를 열면 가장 먼저 보이는 **큰 문장**입니다. 언제든 원하는 대로 바꿀 수 있어요.
아래 순서만 따라 하면 됩니다. 프로그래밍 지식은 필요 없습니다.

---

## 1. 어디를 고치나요

저장소의 **`index.html`** 파일을 엽니다. (메모장·VS Code·GitHub 웹 편집기 등 아무 텍스트 편집기)

파일에서 **`히어로 헤드라인`** 이라는 글자를 찾으세요. (편집기의 "찾기" 기능: `Ctrl+F` / `⌘+F` → `히어로 헤드라인` 입력)

이런 블록이 나옵니다:

```html
<h1 class="cr-kick">
<span class="lead">잘 만든 게임을 문서로 뜯어보고,<br>뜯어낸 문법으로 <em>제 게임을 만듭니다</em></span>
<span class="who">배현 · 게임 기획</span>
</h1>
```

이 **두 줄만** 고치면 됩니다. 나머지는 건드리지 마세요.

---

## 2. 무엇을 고치나요

| 줄 | 무엇 | 바꾸는 부분 |
|---|---|---|
| `<span class="lead"> … </span>` | 큰 문장 | `…` 자리의 글자 |
| `<span class="who"> … </span>` | 작은 이름·직군 줄 | `…` 자리의 글자 |

**`<span class="lead">` 와 `</span>` 사이의 글자만** 바꾸면 됩니다. 앞뒤 `<span …>`, `</span>` 는 그대로 두세요.

### 두 가지 꾸미기 도구

- **줄 바꾸기** → 바꾸고 싶은 자리에 `<br>` 를 넣습니다.
- **주황색 강조** → 강조할 말을 `<em>` 과 `</em>` 로 감쌉니다.

예를 들어:

```
<em>전략을 만들게</em>
```

이렇게 하면 "전략을 만들게" 부분만 주황색으로 강조됩니다.

---

## 3. 예시 — 이렇게 바꿀 수 있어요

**예시 A — 짧고 강하게**
```html
<span class="lead">규칙으로 설계하고,<br><em>명세</em>로 전달합니다</span>
<span class="who">배현 · 게임 기획</span>
```

**예시 B — 한 문장으로**
```html
<span class="lead">플레이가 <em>전략</em>이 되도록 설계합니다</span>
<span class="who">배현 · 게임 기획</span>
```

**예시 C — 나를 소개**
```html
<span class="lead">문제를 찾고, <em>구조</em>로 답하는<br>게임 기획자입니다</span>
<span class="who">배현 · Game Designer</span>
```

**예시 D — 강조 없이 담백하게** (`<em>` 안 써도 됩니다)
```html
<span class="lead">플레이어의 선택을 설계합니다</span>
<span class="who">배현 · 게임 기획</span>
```

> 문장은 1~2줄이 가장 보기 좋습니다. 너무 길면 첫 화면을 넘어갈 수 있어요.

---

## 4. 저장하고 반영하기

1. 고친 뒤 파일을 **저장**합니다.
2. 저장소에 **올리면**(commit → push) 1~2분 뒤 https://00ny.github.io 에 자동으로 반영됩니다.
   - GitHub 웹에서 편집했다면 "Commit changes" 버튼만 누르면 됩니다.

---

## 5. 주의사항 (딱 3가지)

- ✅ `<span class="lead">` **와** `</span>` **사이의 글자만** 바꾸기
- ⚠️ 꺾쇠괄호 `<`, `>` 와 따옴표 `"` 는 지우지 말기 — 화면이 깨질 수 있어요
- ⚠️ 태그는 **짝**으로 — `<em>` 을 열었으면 꼭 `</em>` 로 닫기, `<br>` 는 닫지 않아도 됩니다

혹시 깨졌다면, 위 3장의 예시 중 하나를 **그대로 복사해 붙여넣으면** 원래대로 돌아옵니다.
